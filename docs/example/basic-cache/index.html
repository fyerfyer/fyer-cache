Simple Cache Examples

基本内存缓存​

memory_cache.go 示例演示了 FyerCache 内存缓存的基本操作：

  • 创建内存缓存
  • 存储和检索字符串类型数据
  • 存储和检索结构体类型数据
  • 删除缓存项
  • 设置过期时间
  • 自定义淘汰回调

关键代码段：

// 创建内存缓存
memCache := cache.NewMemoryCache()

// 设置字符串值，带1分钟过期时间
memCache.Set(ctx, "user:1001", "John Doe", 1*time.Minute)

// 设置结构体值，带2分钟过期时间
user := User{
    ID:    1002,
    Name:  "Jane Smith",
    Email: "jane@example.com",
}
memCache.Set(ctx, "user:1002", user, 2*time.Minute)

// 获取缓存值
val, err := memCache.Get(ctx, "user:1001")

// 删除缓存项
memCache.Del(ctx, "user:1001")

这个示例展示了基本的 CRUD 操作以及过期时间设置，是使用 FyerCache 的入门示例。

内存受限缓存​

memory_constrained_cache.go 示例演示了如何创建和使用内存限制的缓存：

  • 设置内存使用上限
  • 基于 LRU 策略自动淘汰缓存项
  • 监听缓存项淘汰事件
  • 观察不同类型数据对内存使用的影响

关键代码段：

// 创建淘汰回调
memCache := cache.NewMemoryCache(
    cache.WithEvictionCallback(func(key string, value any) {
        evictedKeys = append(evictedKeys, key)
        fmt.Printf("Item evicted: key=%s\n", key)
    }),
)

// 创建内存限制为 5KB 的缓存
memoryLimitBytes := int64(5 * 1024) // 5KB
constrainedCache := cache.NewMemCacheMemory(memCache, memoryLimitBytes)

// 添加数据并查看内存使用情况
data := make([]byte, 1000+i)
constrainedCache.Set(ctx, key, data, 10*time.Minute)
memUsage := constrainedCache.MemoryUsage()

这个示例特别有用于那些需要控制缓存内存使用量的场景，确保缓存不会无限增长消耗系统资源。

分片缓存示例​

sharding_cache.go 示例展示了如何使用分片来提高缓存的并发性能：

  • 创建默认分片数的内存缓存
  • 执行基本的缓存操作
  • 演示分片在高并发下的性能优势
  • 比较单分片与多分片缓存的性能差异

关键代码段：

// 创建默认分片数的缓存
defaultCache := cache.NewMemoryCache()

// 对比单分片和多分片缓存在高并发下的性能
singleShardCache := cache.NewMemoryCache(cache.WithShardCount(1))
multiShardCache := cache.NewMemoryCache() // 默认分片数

分片缓存通过减少锁争用显著提高了并发访问性能，特别适用于高并发读写场景。

运行示例​

要运行这些示例，您可以使用以下命令：

# 运行基本内存缓存示例
go run example/basic/memcache/memory_cache.go

# 运行内存受限缓存示例
go run example/basic/mem_constrained_cache/memory_constrained_cache.go

# 运行分片缓存示例
go run example/shard/sharding_cache.go
