Config Options

本文档详细介绍了 FyerCache 中所有可用的配置选项，按组件类别进行组织。

内存缓存选项​

MemoryCache 的选项用于配置内存缓存的行为，如清理间隔、分片数量和回调函数等。

选项描述默认值示例用法
WithEvictionCallback设置当缓存项被驱逐时调用的回调函数无WithEvictionCallback(func(key string, value any) { log.Printf("Evicted: %s", key) })
WithCleanupInterval设置清理过期项的间隔时间2 分钟WithCleanupInterval(5 * time.Minute)
WithShardCount设置缓存内部分片数量，提高并发性能32WithShardCount(64)
WithAsyncCleanup启用异步清理机制，降低清理对主线程的影响falseWithAsyncCleanup(true)
CacheWithWorkerCount设置异步清理工作协程数量CPU 核心数CacheWithWorkerCount(8)
CacheWithQueueSize设置异步清理任务队列大小128CacheWithQueueSize(256)

示例：创建内存缓存​

cache := cache.NewMemoryCache(
    cache.WithCleanupInterval(1 * time.Minute),
    cache.WithShardCount(64),
    cache.WithAsyncCleanup(true),
    cache.WithEvictionCallback(func(key string, value any) {
        log.Printf("Key evicted: %s", key)
    }),
)

清理器选项​

Cleaner 的选项用于配置缓存清理器的行为，主要用于异步清理模式。

选项描述默认值示例用法
WithCleanerWorkerCount设置清理器工作协程数量4WithCleanerWorkerCount(8)
WithCleanerInterval设置清理间隔时间2 分钟WithCleanerInterval(30 * time.Second)
WithCleanerQueueSize设置清理任务队列大小128WithCleanerQueueSize(256)
WithCleanerEvictionCallback设置清理器的淘汰回调函数无WithCleanerEvictionCallback(onEvict)

示例：创建清理器​

cleaner := cache.NewCleaner(
    shardedMap,
    cache.WithCleanerWorkerCount(8),
    cache.WithCleanInterval(30 * time.Second),
    cache.WithQueueSize(256),
)

分片缓存选项​

NodeShardedCache 的选项用于配置基于节点的分片缓存，主要涉及一致性哈希参数。

选项描述默认值示例用法
WithNodeHashReplicas设置每个物理节点的虚拟节点数量100WithNodeHashReplicas(200)
WithNodeCustomHashFunc设置自定义哈希函数CRC32WithNodeCustomHashFunc(fnvHash)
WithNodeReplicaFactor设置数据复制的副本因子（每个键复制到多少个节点）1WithNodeReplicaFactor(2)

示例：创建分片缓存​

shardedCache := cache.NewNodeShardedCache(
    cache.WithNodeHashReplicas(200),
    cache.WithNodeReplicaFactor(3),
)

// 添加节点
shardedCache.AddNode("node1", node1Cache, 1)  // 权重为1
shardedCache.AddNode("node2", node2Cache, 2)  // 权重为2

集群节点选项​

Node 的选项用于配置集群节点的行为，包括通信参数、时间间隔等。

选项描述默认值示例用法
WithNodeID设置节点唯一标识符无WithNodeID("node-1")
WithBindAddr设置节点绑定地址":7946"WithBindAddr("192.168.1.100:7946")
WithGossipInterval设置 Gossip 协议通信间隔1 秒WithGossipInterval(500 * time.Millisecond)
WithSyncInterval设置数据同步间隔30 秒WithSyncInterval(10 * time.Second)
WithProbeInterval设置节点健康检查间隔1 秒WithProbeInterval(2 * time.Second)
WithProbeTimeout设置健康检查超时时间500 毫秒WithProbeTimeout(1 * time.Second)
WithSuspicionMult设置可疑状态乘数3WithSuspicionMult(5)
WithSyncConcurrency设置数据同步并发数2WithSyncConcurrency(4)
WithVirtualNodes设置虚拟节点数量100WithVirtualNodes(200)
WithMaxRetries设置最大重试次数3WithMaxRetries(5)
WithRetryInterval设置重试间隔200 毫秒WithRetryInterval(100 * time.Millisecond)
WithEventBufferSize设置事件通道缓冲区大小100WithEventBufferSize(200)

示例：创建集群节点​

node, err := cluster.NewNode(
    "node-1",
    localCache,
    cluster.WithBindAddr("192.168.1.100:7946"),
    cluster.WithGossipInterval(500 * time.Millisecond),
    cluster.WithProbeInterval(1 * time.Second),
    cluster.WithSuspicionMult(3),
    cluster.WithEventBufferSize(100),
)

分布式缓存选项​

DistributedCache 的选项用于配置分布式缓存的行为。

选项描述默认值示例用法
WithNodeID设置节点唯一标识符无WithNodeID("dist-node-1")
WithAddress设置节点地址无WithAddress("192.168.1.100:8080")
WithWeight设置节点权重1WithWeight(2)
WithReplicaFactor设置复制因子1WithReplicaFactor(3)
WithVirtualNodes设置一致性哈希环中的虚拟节点数量100WithVirtualNodes(200)
WithOperationTimeout设置操作超时时间2 秒WithOperationTimeout(1 * time.Second)
WithRetryOptions设置重试参数最大重试 3 次，间隔 100msWithRetryOptions(5, 200 * time.Millisecond)

示例：配置分布式缓存​

config := distributed.NewDistributedCacheConfig()
config = distributed.WithNodeID("node-1")(config)
config = distributed.WithAddress("192.168.1.100:7946")(config)
config = distributed.WithReplicaFactor(2)(config)
config = distributed.WithOperationTimeout(3 * time.Second)(config)

远程缓存选项​

RemoteCache 的选项用于配置与远程缓存节点的连接。

选项描述默认值示例用法
WithTimeout设置远程操作超时时间2 秒WithTimeout(500 * time.Millisecond)
WithRetry设置重试参数最大重试 3 次，间隔 100msWithRetry(5, 200 * time.Millisecond)
WithDebugLog启用调试日志falseWithDebugLog(true)

示例：创建远程缓存客户端​

remoteCache := distributed.NewRemoteCache(
    "remote-node-1",
    "cache.example.com:8080",
    distributed.WithTimeout(500 * time.Millisecond),
    distributed.WithRetry(3, 100 * time.Millisecond),
    distributed.WithDebugLog(true),
)

指标监控选项​

MonitoredCache 的选项用于配置缓存指标监控。

选项描述默认值示例用法
WithMetricsServer配置指标 HTTP 服务器地址":8080"WithMetricsServer(":9090")
WithMetricsPath设置指标路径"/metrics"WithMetricsPath("/custom/metrics")
WithCollectInterval设置指标收集间隔1 分钟WithCollectInterval(30 * time.Second)
WithMemoryUpdateInterval设置内存使用统计更新间隔1 分钟WithMemoryUpdateInterval(30 * time.Second)
WithNamespace设置指标命名空间"fyercache"WithNamespace("myapp")
WithSubsystem设置指标子系统名称"cache"WithSubsystem("redis")
WithEnableMemStats启用或禁用内存统计trueWithEnableMemStats(true)
WithLabels添加自定义标签无WithLabels(map[string]string{"env": "prod"})

示例：创建监控装饰器​

monitoredCache := metrics.NewMonitoredCache(
    originalCache,
    metrics.WithMetricsServer(":8080"),
    metrics.WithCollectInterval(1 * time.Minute),
    metrics.WithLabels(map[string]string{
        "service": "user-service",
        "env":     "production",
    }),
)

API 服务器选项​

APIServer 的选项用于配置 API 服务器的行为。

选项描述默认值示例用法
WithBindAddress设置服务器绑定地址":8080"WithBindAddress(":9000")
WithBasePath设置 API 基础路径"/api"WithBasePath("/cache/api")
WithTimeouts设置所有超时参数读/写 10s，空闲 60sWithTimeouts(5*time.Second, 5*time.Second, 30*time.Second)
WithReadTimeout设置读取超时10 秒WithReadTimeout(5 * time.Second)
WithWriteTimeout设置写入超时10 秒WithWriteTimeout(5 * time.Second)

示例：创建 API 服务器​

apiServer := api.NewAPIServer(
    cache,
    api.WithBindAddress(":8081"),
    api.WithBasePath("/cache/api"),
    api.WithTimeouts(5*time.Second, 5*time.Second, 30*time.Second),
)

一致性策略选项​

用于配置缓存一致性策略，包括 Cache Aside、Write Through 和基于消息队列的通知策略。

Cache Aside 选项​

选项描述默认值示例用法
WithCacheAsideTTL设置缓存默认 TTL5 分钟WithCacheAsideTTL(10 * time.Minute)
WithCacheAsideWriteOnMiss设置未命中时是否写入缓存trueWithCacheAsideWriteOnMiss(false)

Write Through 选项​

选项描述默认值示例用法
WithWriteThroughTTL设置缓存默认 TTL5 分钟WithWriteThroughTTL(10 * time.Minute)
WithDelayedWrite启用延迟写入及设置延迟时间false, 100msWithDelayedWrite(true, 100 * time.Millisecond)

MQ 通知器选项​

选项描述默认值示例用法
WithMQTopic设置消息主题"cache:consistency"WithMQTopic("cache:invalidation")
WithMQRetry设置重试策略3 次，无延迟WithMQRetry(5, 100 * time.Millisecond)
WithMQNotifierTTL设置缓存默认 TTL5 分钟WithMQNotifierTTL(10 * time.Minute)

示例：创建一致性策略​

// Cache Aside 策略
cacheAsideStrategy := consistency.NewCacheAsideStrategy(
    cache,
    dataSource,
    consistency.WithCacheAsideTTL(10 * time.Minute),
    consistency.WithCacheAsideWriteOnMiss(true),
)

// Write Through 策略
writeThroughStrategy := consistency.NewWriteThroughStrategy(
    cache,
    dataSource,
    consistency.WithWriteThroughTTL(10 * time.Minute),
    consistency.WithDelayedWrite(true, 100 * time.Millisecond),
)

// MQ 通知器策略
mqStrategy := consistency.NewMQNotifierStrategy(
    cache,
    dataSource,
    mqPublisher,
    consistency.WithMQTopic("cache:invalidation"),
    consistency.WithMQNotifierTTL(10 * time.Minute),
)

复制选项​

用于配置复制系统的选项，适用于 Leader 节点和 Follower 节点。

选项描述默认值示例用法
WithNodeID设置节点 ID无WithNodeID("repl-node-1")
WithAddress设置节点地址无WithAddress("localhost:8080")
WithRole设置节点角色RoleFollowerWithRole(replication.RoleLeader)
WithHeartbeatInterval设置心跳间隔500 毫秒WithHeartbeatInterval(1 * time.Second)
WithElectionTimeout设置选举超时1 秒WithElectionTimeout(2 * time.Second)
WithMaxLogEntries设置单次同步最大日志条目数1000WithMaxLogEntries(2000)
WithSnapshotThreshold设置触发快照的日志条目阈值10000WithSnapshotThreshold(20000)
WithMaxRetries设置最大重试次数3WithMaxRetries(5)
WithRetryInterval设置重试间隔200 毫秒WithRetryInterval(100 * time.Millisecond)
WithReplicationTimeout设置复制超时5 秒WithReplicationTimeout(10 * time.Second)
WithPriority设置选举优先级0WithPriority(10)

示例：创建复制节点​

// 创建主节点
leaderNode := replication.NewLeaderNode(
    "leader-1",
    cache,
    log,
    syncer,
    replication.WithHeartbeatInterval(500 * time.Millisecond),
    replication.WithMaxRetries(3),
    replication.WithPriority(10),
)

// 创建从节点
followerNode := replication.NewFollowerNode(
    "follower-1",
    cache,
    log,
    syncer,
    replication.WithElectionTimeout(2 * time.Second),
    replication.WithMaxLogEntries(1000),
)

集成选项​

分片集群缓存选项​

用于配置分片集群缓存的行为。

选项描述默认值示例用法
WithVirtualNodeCount设置一致性哈希环中的虚拟节点数量100WithVirtualNodeCount(200)
WithReplicaFactor设置数据复制的副本因子1WithReplicaFactor(2)

主从集群选项​

用于配置主从架构集群的行为。

选项描述默认值示例用法
WithSyncerAddress设置同步器地址无WithSyncerAddress("localhost:8080")
WithInitialRole设置初始角色RoleFollowerWithInitialRole(replication.RoleLeader)
WithElectionTimeout设置选举超时1 秒WithElectionTimeout(2 * time.Second)
WithHeartbeatInterval设置心跳间隔500 毫秒WithHeartbeatInterval(1 * time.Second)

示例：创建集成缓存​

// 创建分片集群缓存
shardedClusterCache, err := integration.NewShardedClusterCache(
    localCache,
    "node-1",
    "localhost:7946",
    []cluster.NodeOption{
        cluster.WithGossipInterval(500 * time.Millisecond),
    },
    []integration.ShardedClusterCacheOption{
        integration.WithVirtualNodeCount(200),
        integration.WithReplicaFactor(2),
    },
)

// 创建主从集群缓存
leaderFollowerCluster, err := integration.NewLeaderFollowerCluster(
    "node-1",
    "localhost:7946",
    localCache,
    []cluster.NodeOption{
        cluster.WithGossipInterval(500 * time.Millisecond),
    },
    []integration.LeaderFollowerClusterOption{
        integration.WithSyncerAddress("localhost:8080"),
        integration.WithInitialRole(replication.RoleLeader),
    },
)

组合使用选项​

FyerCache 的各种选项可以组合使用，创建高度定制化的缓存实例。以下是一个组合多种选项的示例：

// 创建基础内存缓存
memCache := cache.NewMemoryCache(
    cache.WithShardCount(64),
    cache.WithCleanupInterval(30 * time.Second),
    cache.WithAsyncCleanup(true),
)

// 添加监控装饰器
monitoredCache := metrics.NewMonitoredCache(
    memCache,
    metrics.WithMetricsServer(":8080"),
    metrics.WithLabels(map[string]string{"env": "production"}),
)

// 创建分布式集群缓存
clusterCache, err := cluster.NewClusterCache(
    monitoredCache,
    "node-1",
    ":7946",
    cluster.WithGossipInterval(500 * time.Millisecond),
    cluster.WithProbeInterval(1 * time.Second),
)

// 创建 API 服务器
apiServer := api.NewAPIServer(
    clusterCache,
    api.WithBindAddress(":8081"),
    api.WithBasePath("/cache/api"),
)
